# TB Transcriptomic Benchmark – Analysis Scripts

This repository contains the analysis scripts used for the manuscript:

"Cross-Cohort Benchmarking of Model Architectures and Published Gene Panels for Whole-Blood Tuberculosis Transcriptomic Classification Across Disease States."

Processed datasets:
https://doi.org/10.5281/zenodo.18923347

Main scripts:

Model Internal.py – LOCO internal validation  
Model External.py – External cohort testing  
Model Significance testing.py – Bootstrap ΔAUC model comparisons  

Panel Internal.py – Internal panel evaluation  
Panel External.py – External panel evaluation  
Panel Significance testing.py – Bootstrap ΔAUC panel comparisons  

PCA analysis.py – PCA domain shift analysis  
Final AUC Plots.py – Figure generation
